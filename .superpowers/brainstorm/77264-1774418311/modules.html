管理界面功能模块

请选择第一期要包含的模块（可多选）

1

Dashboard — 概览

API Keys 总数、后端健康状态、今日请求量、今日费用、最近请求列表。只读，无操作。

2

API Keys 管理

列表（含用量/预算）、创建（随机生成，明文一次性展示）、编辑速率/预算/状态、停用/删除。这里解决「key 明文一次性展示」问题。

3

Backends 配置

列表后端（Bedrock/Gemini/Anthropic/OpenAI）、新增/编辑（表单明文输入凭证，保存时加密写库）、启用/禁用、健康状态显示。这里解决「加密存储」问题。

4

Model Mappings 模型映射

查看/编辑 source → target 模型映射规则，设置优先级、价格、provider。支持通配符。

5

Usage 用量统计

按 API Key / 时间段查询请求记录、Token 消耗、费用汇总，图表展示。复用现有 /v1/usage API。

6

Feature Flags 功能开关

查看/切换系统功能开关（tool_use、caching、ptc 等）。简单的开关列表。

点击可多选，选好后在终端告诉我你的决定
